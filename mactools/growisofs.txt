A binary for growisofs is included in HomeBrew package dvd+rw-tools.

Install HomeBrew:
  /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Install dvd+rw-tools:
  brew install dvd+rw-tools

Binary for growisofs is at:
  /usr/local/Cellar/dvd+rw-tools/<version>/bin/growisofs
